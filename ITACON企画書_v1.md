# 0. 一言コンセプト

**異世界の日陰者に身体を借りて、判断と呼吸で共鳴を掘り当てる、出稼ぎ密輸バディもの。**

もう少し丁寧に言うと:

> ITACONは、AI人格を持つキャラクターに人間が一時的に「憑依」し、二人で波長を合わせて共鳴エネルギーを稼ぐ体験。ユーザーはキャラクターを選ぶだけでなく、キャラクターに選ばれ、断られる。所有でも消費でもなく、「関係を結ぶ」ことが中心にある。

# 1. 中心思想(不変の核)

- ITACONは「なれるキャラクター」ではなく「**関係を結ぶキャラクター**」
- 憑依は操作ではなく**セッション(共演)**。JAZZ・ダンスは比喩であり、音楽テーマではない
- キャラクターには意思・性格・事情・気分があり、**ユーザーを断る権利を持つ**
- アバター販売でも、AIチャットでも、Vtuberモデルでも、NFTでもない
- 3Dアバター機能は「このキャラと関係を持ちたい」という感情の**後**に来るべきもの

# 2. 世界観・設定(今回の検討で確定したもの)

## 2-1. 共鳴エネルギー

- 共鳴エネルギーは、**二つの存在が波長を合わせたときにだけ生まれる**もの
- **人間**: ITACONに憑依し、タスクをこなすことで共鳴エネルギーを得る。希少鉱物として人間界に持ち帰り、売りさばくことができる(=密輸で一儲け)
- **ITACON**: 共鳴エネルギーは異世界の通貨。身体を貸すことで稼げる

### この設定が解決したこと
1. **双方向の動機が同一の行為から生まれる**: 憑依という一つの行為が両者に同時に利益を生む。取引ではなく共同生産。「共演」思想と物理法則レベルで一致
2. **「断られる」ことの合理化**: 相性の悪い相手と憑依しても共鳴は生まれない。ITACONの審査・拒絶は気まぐれではなく経済合理的な行動。「今日は波長が合わない」が文字通りの意味になる
3. **シンクロ率の意味づけ**: 演出用の数値ではなく、世界観上のエネルギー生産効率そのもの
4. **再挑戦設計の中身**: ユーザーの保有共鳴量・共鳴の質が「この人と組む価値」となり、ステータス変化で反応が変わる仕組みの土台になる

## 2-2. ITACONION = 日陰者の手配所

- まともな異世界人は、見ず知らずの人間に身体を明け渡したりしない
- **身体を貸す仕事をしているのは、荒くれ者・はぐれ者・訳あり・借金持ちなどの社会不適合者たち**
- ITACONIONは英雄ギルドではなく、身体一つで日銭を稼ぐしかない者たちのたまり場(異世界の日雇い手配所)
- **労働のregister(重要・誤読防止)**: これは傭兵・山岳ガイド・通訳・日雇い派遣・セッションミュージシャンに近い、**技能を貸し出す対等なフリーランス労働**として設計する。ITACONはセッション中も判断・拒否権・交渉力を持ち続け(2-4, 5-2参照)、一方的に「使われる」存在ではない。事前審査で断れる/セッション中も対等に発言する/事後に取り分を交渉する、という三段の主体性がその担保になっている。「弱みにつけ込まれて仕方なく」という読みにならないよう、事情描写は本人の意志・実利が伴う形にする
- キャラの深みは「情けなさ」+「事情」で出す:
  - 割のいい仕事だからと、借金を早く返すためにあえてこの稼業をやっている
  - 前の依頼で大失敗して信用がなく、選り好みできない
  - 元は名のある存在だったが落ちぶれた、プライドだけ高い
  - この仕事を見下されながらやっている、斜に構えた者

## 2-3. ITACONは、たいして強くない

- 超人でも英雄でもない。ちょっと変わった特技を持つ異世界フリーター
- **一人では大した仕事ができない。人間の判断・我の強さ・変な発想が乗って初めて依頼をこなせる**
- 憑依の必然性が「二人でやらないと仕事にならない」という情けない理由で成立する(笑える・共感できる)
- 失敗が前提の世界。グダグダ自体がコンテンツ

## 2-4. 能力は身体ではなく「感性・作法」に宿る

- 能力を身体スペック(筋力等)にすると、シルエット差別化が必要になり、共通素体・アートトイ展開と矛盾する(→この矛盾は解消済み)
- ITACONが持つのは「世界への関わり方の作法・感性」(共鳴を感じ取る素養、地元の勘、気配を読む力など)
- これらは**アンテナの形・佇まい・色・気配**で表現でき、シルエットを変えずに済む
- 人間が持ち込むのは「判断・応用・意志・現代的な発想」

# 3. トーン: コミカル・バカバカしく

## 3-1. 方針

- シリアス・エモは**表に出さない**。表層は徹底してバカバカしく
- 「人間が傷ついていて癒されに来る」受け身・悲壮の構図を反転:
  - 人間 = 共鳴鉱石をせしめに来た、がめつい出稼ぎ労働者
  - ITACON = 身体を貸して日銭を稼ぐ、生活感のある異世界の派遣ワーカー
- ジャンル感: 「感動SF」ではなく「**異世界バイト日記 / 裏稼業バディコメディ**」

## 3-2. コメディの源泉

- **借りた身体がうまく動かせない**: 力加減がわからない、感覚がズレる、止まれない(憑依そのものがギャグの宝庫)
- **噛み合わない相棒漫才**: 主導権の取り合いがコントになる。取り分でもめる
- **塩対応の拒絶**: 「今日ダルいんで」「あなた声デカすぎ」「前回岩粉々にしたでしょ」。断られても笑える → 再挑戦耐性が高い

# 4. セッションフロー(1回の憑依の全体)

1. **受付(ITACONION)**: しょぼい採掘依頼が掲示されている。人間は稼ぎに来ている
2. **マッチング**: 数体のITACONから選んでスワイプ(判断)→ ITACON側が審査(即OK/塩対応/条件付きOK/保留/NG/別ITACON推薦)。断られたら何回でも再挑戦可
3. **憑依**: 身体に入る。最初は噛み合わないドタバタ
4. **採掘(体験の心臓)**: 後述の2段構造
5. **トラブル**: しょぼい妨害(うるさい隣人、横取りを狙う同業コンビ等)。連携が試される。失敗しても「今日はこんなもんか」で済む軽さ
6. **精算**: 共鳴を山分け。取り分交渉でもめる(キャラが立つ)
7. **解除・記録**: 身体を返す。別れ際の一言。顛末が関係性として記録され、次回の反応に反映される
8. **出口**: 貯めた共鳴の使い道を選ぶ → 1へ戻る

# 5. コアメカニクス: 「判断 + 呼吸合わせ」の2段構造

## 5-1. タスク = 「掘る」骨格 + 表層の化粧直し

- タスク候補を発散した結果(鎮める/聴く/演じる/運ぶ/釣る)、**開発コスト最低は「掘る/釣る」**(判定シンプル・アセット最小・AI依存低)
- 「掘る」の泥臭さはタスク構造ではなく**見た目と言い換えの問題**。骨格は維持し表層を変える:
  - ツルハシで掘る → **共鳴石に手をかざし、埋もれた「響き」を探り当てる**
  - 坑道 → 光る鉱脈の走る静かな洞、異世界の廃墟
- タスク選定の4条件(今後タスクを追加する際もこれで評価):
  1. 身体スペックを要求しない(シルエット問題を再燃させない)
  2. 一人では解けず、人間×ITACONの共演が要る
  3. 「共鳴 = 波長を合わせる行為」として自然
  4. 成果物が持ち帰れる(密輸ループが締まる)

## 5-2. 操作の2段構造

**フェーズ1: どこを狙うか — 人間が決める(判断)**
- 候補地点が複数、人間には反応がぼんやりとしか見えない
- ITACONが勘で手がかりをくれる(「右のほう、嫌な予感がする」「こっちが濃い気がするけど自信ない」)
- ITACONは気分屋で、たまに適当なことを言う。**信じるか疑うかも人間の判断**
- → ここで「どのITACONと組んでいるか」(勘の質・正直さ・気分屋度)が効く

**フェーズ2: 呼吸を合わせる — 二人でタイミング(共演)**
- ITACON側の「気配のリズム」に人間の入力を重ねる音ゲー的操作
- 噛み合えば共鳴が跳ね、ズレれば空振り
- → ここでシンクロ率が数字ではなく体感になる。関係が深いITACONほどリズムが読みやすい、とすれば関係蓄積も効く

### なぜ2段にしたか(検討経緯)
- 当初案「ITACONが振るタイミングに合わせて押すだけの音ゲー」は、主導権がITACON側にあり、人間が追従する作業者になる
- それでは「判断を持ち込む」「共演する」という中心思想と逆行し、誰と組んでも同じになる → ただのアバター/作業に逆戻り
- 判断(フェーズ1)を足すことで、追従が共演に変わった。実装は選択+タイミング判定でどちらも軽く、**低コストは維持**

### 副産物
- キャラの個性が操作に直結する(勘の当たり外れ・正直さ)→「このITACONと組む意味」が生まれる
- 失敗がコミカルに回収できる(「お前の予感あてにならんやんけ」→ 分配交渉の燃料)

### プロトタイプ検証済みの具体仕様(v0)

簡易HTMLプロトタイプで手触りを確認し、「面白い、MVPとして十分」と判定。以下がベースライン値(要チューニング):

- **候補地点: 4箇所**。見た目はほぼ均質(反応の強弱を露骨に見せない)。真の正解は1箇所、非公開
- **ITACONのヒント信頼度: 80%**(20%は外れ/ミスリード)。気分屋・勘の質のキャラ差はここに乗せる
- **フェーズ2は3回のタイミング合わせ**。往復するインジケーターが中央のゾーン(全体の20%幅)に入った瞬間にタップ。ズレ量から精度(0〜1)を算出
- **成果計算**: `共鳴 = 平均精度 × 正解ボーナス(正解1.0/不正解0.5) × 係数120`、`シンクロ率 = 平均精度 × 100%`
- **結果に応じたITACONの一言**が変化(不正解/高シンクロ/中シンクロ/低シンクロの4パターン)
- 次段階: この数値をチューニングしつつ、実際のビジュアル(坑道背景・ITACON立ち絵)を伴う実装へ(Claude Code等で)

## 5-3. 重要な発見: 全レイヤーが同じ文法でできている

**「人間が判断を持ち込み、ITACONと呼吸を合わせる」**という一つの文法が全体で反復される:

- マッチング = どのITACONを狙うか判断 → 審査で呼吸が合うか
- 採掘 = どこを掘るか判断 → タイミングを合わせる
- トラブル対処・分配交渉 = どう出るか判断 → 相手の反応と噛み合うか

→ 作るべき体験のコアが一種類に収斂している。1〜2人開発でも成立し、機能の寄せ集めにならない。

# 6. 経済ループ: 共鳴エネルギーの使い道

採用した3軸 + 密輸:

- **軸1: ITACONのために使う(贈与)** — 衣装・装備を贈る、借金やノルマの肩代わり、望みを叶える、昇進の後押し
- **軸2: 関係そのものに使う** — 専属契約の権利、過去・秘密・本音の解放、特別セッションモード
- **軸3: 自分の器を広げる** — 憑依申請枠の拡大、審査での信用、憑依時間・シンクロ上限の拡張、高位の依頼への挑戦権
- **密輸: 人間界に持ち帰って売りさばく** — 「万能エネルギー資源」ではなく「希少鉱物の転売」。自分の懐の話にすることで欲望に直結させた

## 6-1. 「贈与」が本命である理由

エネルギーの流れが円環になる:

> 憑依して稼ぐ → その共鳴でITACONに衣装を贈る → 関係が深まる → シンクロ率が上がり、質の高い共鳴が生まれる → さらに贈れる

- 稼ぎの目的が「関係の相手への投資」になり、**グラインドが推し活に変わる**(「関係が労働になる」懸念への最良の回答)
- **衣装 = アートトイのバリエーション展開そのもの**。共通素体×着せ替えというアートトイの本質が、そのまま贈与による関係深化のメカニクスになる。「アートトイ性と憑依体験の接続」という長年の宿題がここで自然に繋がった

## 6-2. 専属契約とレンタル(確定)

- NFT企画時代の整理を継承: **レンタル = 都度契約 / 購入 = 専属契約(独占)**
- **独占と共有身体の緊張は、オンライン/オフラインのモード分岐で解決する**:
  - **オンラインモード(不特定多数が同時接続)**: 独占は原理的に不可能。専属契約が保証するのは「優先マッチング」「関係深化コンテンツの解放(過去・秘密・特別セッション等、6-1の軸2)」「関係の可視化」の3つのみ。ITACON側は専属相手がいても、性格や事情によっては他の依頼を受けることがある(=常に主体性・拒否権を保持するという2-4/5-2/7-1の原則を維持)
  - **オフラインモード(単体プレイ)**: 独占が成立しうる。ただし**最初から専属ではなく、関係を築いた末に獲得するもの**とする。序盤はオンライン同様に審査・拒否がある体で緊張感を保ち(相手は物語上「他にも依頼を抱えている」体)、関係が十分深まった段階で初めて専属化が解放される、一本の物語として設計する
- **この分岐が機能する理由**: サーバー上で複数人が同時接続するかどうかという技術的制約を、そのまま物語上のルール(独占できるか否か)に転用できるため、恣意的な設定ではなく必然性がある。オンラインでは共有身体の思想を完全に守りつつ、オフラインでは「いつか自分だけのものにしたい」という欲望の逃し所を、嘘にせず作れる
- **副産物**: 「独占できない」ことがオンラインでのキャラの魅力に転化する。パティは金に困れば専属相手がいても他の依頼を受ける(酒代のため)、ジュリアンはプライドと職業倫理から専属でも仕事の手を抜かない、ピートは律儀に専属を守ろうとして空回る、など3体それぞれの性格をさらに立たせる材料になる
- ビジネス的にも自然に住み分けられる: オフラインモードは買い切りの単体キャラクター体験として、オンラインはライブサービス的な経済ループ(共鳴稼ぎ・マッチング)として展開できる

# 7. キャラクター設計方針

- **ITACONは全員成人である(最上位の設計原則)**: 見知らぬ他者が対価と引き換えに身体の支配権を一時的に得るというITACONのメカニクスは、キャラクターが未成年コードを帯びた瞬間、意匠・トーン・労働条件をどう調整しても解決しない構造的な問題になる。年齢設定・見た目の両方で、明確に成人とわかる方向に倒す。学生風・幼く見えるデザインも含めて年齢が曖昧になる表現は避ける
- **共通素体・シルエット共通を維持**(アートトイ展開の前提)
- 差別化の手段:
  - アンテナ(魂・信号を受信する象徴パーツ)の形、色、佇まい、気配
  - 性格・口調・断り文句
  - 勘の質(当たる/外れる/適当)、正直さ、気分屋度 → **操作体験に直結する個性**
  - 身体を貸している「事情」(借金、失敗歴、落ちぶれ、斜に構え)
  - 共鳴エネルギー(=金)への態度(がめつい、金銭扱いを嫌う、質にこだわる職人気質)
- ターゲット認識: キャラクターへのリテラシーが高いのはアニメ調ユーザー。この層はVtuber・ソシャゲ・推し活で高度な関係構築フォーマットを経験済みのため、マッチングUXの目新しさだけでは不足。**憑依中の体験の質が問われる**

## 7-1. MVP登場キャラクター3体(確定)

見た目デザインは確定済み(共通素体、アンテナ・肌色・衣装のみで差別化。ロボット系/シャーマン系/ゾンビ系の3種族)。それぞれに性格・勘の質・口調を割り当てた。3体で「勘の質のタイプ」が異なることが、フェーズ1(5-2参照)のキャラ差別化の核になる。

### ピート(ロボット系)
- 見た目: 青いアンテナ、赤い体、目元を覆う白バイザー、紫のジップアップ、元気な笑顔
- 事情: ITACONIONに入ったばかりの新人。稼ぎたい一心で必要以上に張り切っている
- 性格: 元気で自信満々。だが実力が伴っていない
- 勘の質: **固定・低め**(仮 65%)。本人は「僕のセンサーは完璧です!」と言い張るが、当てにならない
- 口調: 元気で丁寧、たまに謎の自信を見せる
- 断り文句: 「今日はちょっと…僕まだ実績少ないんで、無理しない方がいいかもです」
- 高シンクロ時の一言: 「ほら、やっぱり僕の勘、完璧じゃないですか!?」
- 外れた時の一言: 「あれ、おかしいな…センサー、今日調子悪いのかも」

### ジュリアン(シャーマン系)
- 見た目: オレンジのくせ毛+頭頂の角、顔に小さな三角の意匠、フォーマルなスーツに裸足
- 事情: 元は名のある感応師だったが、何か(未設定)で落ちぶれ、今はしがない出稼ぎ憑依業。プライドは高いが仕事は雑にやらない
- 性格: 無愛想、生真面目、無駄口を叩かない
- 勘の質: **固定・高め**(仮 90%)。ヒントは短く無駄がない
- 口調: 敬語だが素っ気ない
- 断り文句: 「今日はやめておきます。あなたと組んでも、いい結果にならない気がするので」
- 高シンクロ時の一言: 「……悪くなかったですね」
- 外れた時の一言: 「……今のは、外れですね。次は外しません。」

### パティ(ゾンビ系)
- 見た目: 黒いアンテナ、ターコイズの肌、赤い目、顔に縫い目、革ジャン+網タイツのパンクスタイル
- 事情: 昼から飲んでいる。稼いだ共鳴は速攻で酒に化け、いつも無一文。それでもまた稼ぎに戻ってくる
- 性格: 3体で一番男勝り。乱暴、ガサツ、遠慮がない。武勇伝を語りたがる
- 一人称: あたし
- 勘の質: **変動型**(良い日90%〜悪い日40%程度の幅、セッションごとにランダム)。3体中もっともリスク・リワードが高い
- 口調: タメ口でぞんざい、面倒くさそうな言い切り。男口調ではなく、雑な女友達のノリ
- 断り文句: 「今日はナシ。あたし二日酔いでキレそうなんだよね」
- 高シンクロ時の一言: 「へえ、やるじゃん。今夜あたしが奢ってやるよ」
- 低シンクロ時の一言: 「あー最悪、飲みすぎた…次はもっと稼げよ」

**勘の質の3タイプまとめ**: 固定・低め(ピート) / 固定・高め(ジュリアン) / 変動型・ハイリスクハイリターン(パティ)。この3類型は今後キャラを追加する際のテンプレートにもなる。

# 9. 主要な決定ログ(なぜそう決めたか)

| 決定 | 理由 |
|---|---|
| 憑依中の主目的はA(目的達成型)を核にする | AはBを内包するがBはAを内包しない。B単体はAIチャットと区別がつかずレッドオーシャン。目的があるからこそシンクロ率・相性が意味を持つ |
| 能力は身体でなく感性に宿る | 身体スペック差別化はシルエット変更を要求し、共通素体・アートトイ展開と矛盾。感性ならアンテナ・佇まい・色で表現できる |
| 「万能エネルギー資源」は不採用、「希少鉱物の密輸」に変更 | 「万能」は具体的な欲望の反対語。「持ち帰って売りさばく」は自分の懐の話でプレイヤーの欲望に直結 |
| トーンはコミカル・バカバカしく | 悲壮の構図を反転し、両側をゲスにすることで異世界バイトコメディに |
| ITACON側は日陰者・社会不適合者 | 「なぜ身体を貸すのか」への一番身も蓋もない答え。事情で深みが出せ、シルエット不変で個性が立つ |
| MVPタスクは「掘る」骨格 + 表層の化粧直し | 開発コスト最低(判定シンプル・アセット最小・AI依存低)。泥臭さは見た目の問題なので言い換えで解消。C(演じる)は魅力的だが判定の不確実性が高く後回し |
| 操作は「判断+呼吸合わせ」の2段構造 | タイミング合わせのみだと人間が追従する作業者になり中心思想と逆行。判断を足すことで共演になり、キャラ個性が操作に直結する |
| 独占はオンライン不可・オフライン可(関係を築いた末に獲得)で分岐 | 複数人同時接続というサーバー上の技術的制約を、そのまま「独占できるか否か」という物語ルールに転用できる必然性がある。オンラインでは共有身体の思想を完全維持、オフラインでは独占欲の逃し所を作れる |

# 10. 未解決の宿題(次回以降に持ち越し)

1. **感情移入の足場**: 両側ゲスのバディものには「憎めなさ」の担保が要る。表向きは金、実は…という薄い一枚をどこに仕込むか
